🎯에러로그

  1. gitignore 에러
    문제 : 처음에 gitignore없는 상태에서 업로드했다가 뒤늦게 추가했더니
    ignore 된 파일들이 자꾸 표시되는 문제가 발생했습니다.
    해결 : 구글링 결과, git내의 저장된 캐시가 문제가 되어
    이를 삭제하는 다음의 코드를 통해 해결하였습니다.

  2. soup.select(태그) 에러
    문제 : 크롤링할때 설정한 태그명에 맞는 데이터가 출력되지 않고 빈 리스트가 출력되었습니다.
    해결 : 입력한 태그명에 문제가 있다고 판단하여, 하위태그부터 테스트해가며 재설정해주었습니다.
