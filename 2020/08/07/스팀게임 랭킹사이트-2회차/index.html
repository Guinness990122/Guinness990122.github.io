🎯에러로그

  1. vscode pylint import error
    문제 :셀레니움 import 시 pylint(import error 발생)
    해결 :

    위의 명령어로 pylint 설치한 후
    cmd + shift + P 단축키를 통해
    vscode내의 python interpreter을 (가상환경:venv) 가 있는 python 으로 설정했습니다.

  2. parser 설계오류
    문제 :parser에서 초기 데이터 삽입시 각 element를 리스트에 각각 저장하여
    DB에 삽입할 예정이었으나, 3개가 동시에 삽입되지 않기도 하고 로직이 불필요하게 복잡한 것 같았습니다.
    해결 :딕셔너리로 변환한 뒤 이름을 키로, 플레이어수와 최고접속률을 리스트에 넣어 value로 설정한 뒤 DB에 삽입했습니다.
